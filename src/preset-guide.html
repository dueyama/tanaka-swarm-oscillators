Model

Tanaka 縮約モデル

ここでは Dan Tanaka の chemotactic / swarm oscillator を、位置と内部位相をもつ 縮約モデルとして説明します。プリセット名は複数の論文や観察したいパターンに 由来しますが、ブラウザ版では下の方程式と記号を基準に、パラメータ、粒子数、 初期乱数、時間刻み、軌跡設定を整理しています [1-3]。

Shared equations

dψi dt = ∑ j≠i e -|Rji| sin ⁡ ( Ψji + α |Rji| - c1 ) dri dt = c3 ∑ j≠i Rji ^ e -|Rji| sin ⁡ ( Ψji + α |Rji| - c2 )
Rji = rj - ri , Rji ^ = Rji |Rji| , Ψji = ψj - ψi

周期境界では L の箱で minimum image を使い、 自己相互作用 j=i は和から除きます [1-3]。

式の気持ち

位相の式 近くにいる振動子ほど e-|Rji| で強く効きます。括弧の中身 Ψji+α|Rji|-c1 の sine が正なら ψi は進み、負なら遅れます。つまり c1 は、 どの位相差の相手が時計を進めるかを決めるオフセットです。

位置の式 R̂ji は粒子 i から粒子 j へ向かう単位方向です。 sine の値が正なら粒子 i は粒子 j の方向へ動き、負なら逆向きへ動きます。 c2 はその引力・斥力の位相オフセット、 c3 は動く速さ全体の係数です。

距離依存の遅れ α|Rji| は、遠い相手ほど相互作用の位相がずれる項です。 α=0 では相互作用は位相差と c1/c2 だけで決まり、 α を大きくすると clustered clusters のような距離込みの秩序が出やすくなります。

記号とパラメータ

ri
粒子 i の二次元位置。ブラウザでは周期箱の中の座標として描きます。
ψi
粒子 i の内部振動位相。色はこの値を 2π で割ったものに対応します。
Rji
rj-ri。 粒子 i から見た粒子 j への相対ベクトルです。
|Rji|
粒子間距離。相互作用の強さは e-|Rji| で短距離ほど大きくなります。
R̂ji
Rji/|Rji|。 移動方向だけを表す単位ベクトルです。
Ψji
ψj-ψi。 粒子 i から見た粒子 j との位相差です。
c1
位相方程式の応答オフセット。どの位相差の近傍が時計を進めるか、遅らせるかをずらします。
c2
位置方程式の応答オフセット。どの位相差の近傍へ近づくか、離れるかをずらします。
c3
位置ダイナミクスの速度係数。小さいほど位相変化に対して粒子の移動がゆっくりになります。
α
距離に比例する位相遅れ。0 なら距離は強さだけに効き、位相のずれは作りません。
N, L
N は粒子数、L は周期正方形箱の一辺です。
Δt, steps/frame
ブラウザの Euler 積分設定です。論文モデルの物理パラメータではありません。
seed
初期位置と初期位相を決める乱数種。同じプリセットを同じ初期条件から再現するための値です。
trail, velocityClamp
見た目とブラウザ安定性のための実装設定です。Tanaka の方程式そのもののパラメータではありません。

プリセットは入口用の選抜です

Tanaka 系の論文では、membrane / section-of-fruit, clustered clusters, branch, grid with defect, fireworks, train, stick-slip motion, moving membranes など複数の状態が報告・示唆されています [1-3]。 このブラウザ版のプリセットは、その全体を網羅するリストではなく、 軽く動かしながら特徴の違いを掴むための入口です。RIMS phase A-D は、 特に c1 / c2 の相図を読むための代表点として追加しています。 各カードでは論文・資料側の値と、見やすさや速度のために調整したブラウザ値を分けて表示します。

主に動かしているパラメータ

c1
位相応答のずれ。近い相手が時計を進めるか、遅らせるかを変えます。
c2
運動応答のずれ。同じ位相差でも近づく側に出るか、離れる側に出るかを変えます。
c3
移動の速さ。membrane 近傍では小さく、clustered clusters では大きめです。
α
距離に比例する位相遅れ。α=0 では距離は強さだけに効き、大きくすると距離込みの秩序が出やすくなります。

プリセット群の見方

  • Membrane 系小さい c3 と α=0 で、ゆっくり膜・果実断面候補を見る。
  • PRL clustered clustersα と c3 が大きめで、小クラスターの生成と配置を見る。
  • Fireworks explorer論文中に名前が挙がる fireworks / train / stick-slip 系を探すための探索用 candidate。
  • RIMS phase A-Dc1 と c2 の相図上で、位相引力/反発と位置引力/斥力の組み合わせを見る。

RIMS の c1, c2 相図

Tanaka–Iida の RIMS 論文 [2] では、 N=20, L=10, c3=0.02, α=0 を固定し、 c1 と c2 を 0.2 刻みで走査しています。 t=0 から t=5000 は捨て、 t=5000 から t=10000 の時間平均で、 最近接距離の平均と内部状態分散を見ています。

代表点は A: c1=0.5, c2=1.5、B: c1=3.0, c2=2.0、C: c1=6.0, c2=4.0、D: c1=4.0, c2=4.0 です。 ブラウザの A-D プリセットはこの代表点を見やすい粒子数に増やしたものです。
